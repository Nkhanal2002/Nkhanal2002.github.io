Ways to learn Front-End Development

To learn a Front-End Development, one should have a good understanding of three web designing tools. They are:

  1. HTML

    2. HTML stands for Hyper Text Markup Language.It is one of the main three web page designing tool that helps to structure the webpage. For instance, you can think it as a skeleton part of the human body.

  2. CSS

    3. CSS stands for Cascading Style Sheet. It helps to put on some styles on the webpage to make it looks good. For example, you can consider CSS as a human body with just muscles over skeleton part.

  3. JS

    4. JS stands for JavaScript. It is one of the most important tools in building a logical website.For example, you can assume it as a complete human beings with sense organs who can talk and walk.
