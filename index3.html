HTML More About HTML

HTML stands for Hyper Text Markup Language. It is one of the main three web page designing tool that helps to structure the webpage. For instance, you can think it as a skeleton part of the human.

A table showing a few of HTML tags with function and a short example:

HTML Tags Function An Example
<img> To insert image/s to a webpage.
<a> To create a link option in a webpage.
<ul> To create an unorder list.
CSS More About CSS
CSS stands for Cascading Style Sheet. It helps to put on some styles on the webpage to make it looks good. For example, you can consider CSS as a human body with just muscles over skeleton part.
